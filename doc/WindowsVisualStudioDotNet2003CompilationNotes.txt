Compile EMAN2 on Windows with Microsoft Visual Studio .Net 2003
may have these problems:

1. Some Boost Python wrapping files have this compilation error:
 "fatal error C1204: Compiler limit: internal structure overflow" 
 Solutions in the Boost Python FAQ 
 (http://www.boost.org/libs/python/docs/v2/faq.html#c1024):
 Currently, libpyEMData2.cpp and libpyUtils2.cpp have this compilation
 error. 
 
 For libpyEMData2.cpp, comment out these functions:
 done_data()
 update()
 render_amp8()
 set_array_offsets()
 set_array_offsets()
 get_array_oddsets()
 and last three "__call__" functions.
 
 For libpyUtils2.cpp, split this file to libpyUtils2.cpp and otherUtil.cpp. 
 
 2. Manually check All_Build, INSTALL, and test projects in Build->Configuration Manager.
 
 3. move libpy***.dll from /bin to /lib directory.
