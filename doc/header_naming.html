EMAN2  Image Header Attribute Naming Conventions

EMAN2 uses a unique name to reference an image header attribute. The following table is the naming conventions.


Attribute Name
 DataType
 Attribute Description
apix_x
 float
 pixel x size in Å unit 
apix_y
 float
 pixel y size in Å unit
apix_z
 float
 pixel z size in Å unit
nx
 int
 image x-direction size in number of pixels
ny
 int
 image y-direction size in number of pixels
nz
 int
 image z-direction size in number of pixels
origin_row
 float
 origin location along row (x) in number of pixels
origin_col
 float origin location along column (y) in number of pixels
origin_sec
 float origin location along section (z) in number of pixels
minimum
 float minimum pixel density value in the image
maximum
 float maximum pixel density value in the image
mean
 float mean pixel density value in the image
sigma
 float
 sigma  pixel density value in the image
square_sum
 float sum of the squares of each pixel density value in the image
mean_nonzero
 float mean pixel density value in the image, ignoring pixel whose value =0.
sigma_nonzero
 float sigma  pixel density value in the image, ignoring pixel whose value =0.
kurtosis
 float kurtosis pixel density value in the image
skewness
 float skewness pixel density value in the image
datatype int
 pixel data type in the image physical file (e.g., float, int, double, etc). This is an enumeration type.
is_complex int
 1 if this image is a complex image. 0 if it is a real image.
is_ri int
 1 if this is a complex image and is in real/imaginary format; 0 otherwise.
avgnimg
 int
 If the image is the result of averaging, avgnimg is the number of images used in averaging. Otherwise, avgnimg is 0.
dm3_exposure_number
 int
 DM3 image only. exposure number.
dm3_exposure_time
 double
 DM3 image only. exposure time in seconds.
zoom
 double
 zoom
antiblooming
 int
 antiblooming
magnification
 double
 Indicated magnification
frame_type
 string
 frame_type
camera_x
 int DM3 image only. camera size  along x direction in number of pixels.
camera_y
 int DM3 image only. camera size along y direction in number of pixels.
binning_x
 int DM3 image only. Binning #0
binning_y
 int
 DM3 image only. Binning #1
