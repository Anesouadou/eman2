CVS  HowTo

1. set up CVSROOT. For both EMAN and EMAN2, set up the following in your shell startup script:

a) for csh/tcsh
    setenv CVS_RSH ssh
    setenv CVSROOT "blake.3dem.bioch.bcm.tmc.edu:/usr/local/CVS/CVS"

b) for bash/sh/zsh
    export CVS_RSH=ssh
    export CVSROOT="blake.3dem.bioch.bcm.tmc.edu:/usr/local/CVS/CVS"


NOTE: The following supposes you put EMAN2 under $HOME/EMAN2.

2.  To check out EMAN2 code, run
             cd $HOME/EMAN2/src
             cvs co eman2

3.  To check in your files, run
             cd $HOME/EMAN2/src/eman2
             cvs ci YOUR-FILES

      or to check in everything you modified:

             cd $HOME/EMAN2/src/eman2
             cvs ci

4. To merge the CVS code with your code, run
             cd $HOME/EMAN2/src/eman2
             cvs update
