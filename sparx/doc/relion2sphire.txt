===== sxrelion2sparx =====
WARNING!!! sxrelion2sparx.py is obsolete now (2017/12/07). Please use sxrelion2sphire.py instead!  The Wiki page is coming soon.

Converts a Relion star file into an hdf stack with the corresponding header information.

===== Usage =====

Usage in command line

  sxrelion2sparx.py input_star_file --output_dir=output_dir --star_section=star_section --box_size=box_size --create_stack

===== Typical usage =====

So far, sxrelion2sparx will only run in serial mode.

  sxrelion2sparx.py Particles.star --output_dir=newDir --box_size=180 --create_stack

===== Input =====

  ; output_dir : Output directory path.
  ; star_section : Star file selection. This defines the block of the star file which containts the data to be extracted. (default: "data_")
  ; box_size : Box size for particle extraction. It also controls the saved coordinates file format. If the given value is > 0, then the coordinates are stored in eman1 format. If 0, then the sparx format is used. (Default 0: use sparx format).
  ; create_stack : Create particle stack. If set to True, then the script creates a particle stack in hdf format. The image header will contain the CTF and alignment parameters if these were available in the original star file. (default: False)
