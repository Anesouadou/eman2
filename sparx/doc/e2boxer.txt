= Name =
e2boxer - Particle picking: Pick particles with e2boxer.

= Usage =

''usage in command line''

e2boxer.py input --boxsize=BOXSIZE --write_dbbox --write_ptcls --exclude_edges --force --format=FORMAT --norm=NORM --invert --suffix=SUFFIX --dbls=DBLS --autoboxer=AUTOBOXER --ppid=PPID --gui --verbose=n --gauss_autoboxer=GAUSS_AUTOBOXER --do_ctf=DO_CTF --cter --indir=INDIR --nameroot=NAMEROOT --micsuffix=MICSUFFIX --wn=WN --Cs=CS --voltage=VOLTAGE --ac=AC --kboot=KBOOT --debug --apix=APIX

=== Typical usage ===

e2boxer exists in non-MPI version.

    Start the e2boxer GUI with a list of micrographs

    e2boxer.py micrographs_number*


== Input ==

    input:: input images: can not be a list of images: (default required string)

    * The remaining parameters are optional. Normaly you will not touch this.
    boxsize:: box size in pixels: (default int)
    write_dbbox:: write coordinate file (eman1 dbbox) files: (default False)
    write_ptcls:: write particles to disk: (default False)
    exclude_edges:: don't generate output for any particles extending outside the micrograph: (default False)
    force:: force overwrite: (default False)
    format:: format of the output particle images [HDF].: (default string)
    norm:: normalization processor to apply to written particle images: (default string)
    invert:: if writing outputt inverts pixel intensities: (default False)
    suffix:: suffix which is appended to the names of output particle and coordinate files: (default string)
    dbls:: data base list storage, used by the workflow: (default string)
    autoboxer:: a key of the swarm_boxers dict in the local directory, used by the workflow: (default string)
    ppid:: set the PID of the parent process, used for cross platform PPID: (default int)
    gui:: dummy option; used in older version of e2boxer: (default False)
    verbose:: verbose level [0-9], higner number means higher level of verboseness: (default int)
    gauss_autoboxer:: name of autoboxed file whose autoboxing parameters should be used for automatic boxing: (default string)
    do_ctf:: name of file whose ctf estimation parameters should be used for automatic ctf estimation: (default string)
    cter:: CTF estimation using cter: (default False)
    indir:: directory containing micrographs to be processed: (default string)
    nameroot:: prefix of micrographs to be processed: (default string)
    micsuffix:: a string denoting micrograph type: (default string)
    wn:: size of window to use: (default int)
    Cs:: microscope Cs: (default float)
    voltage:: microscope voltage [KV]: (default float)
    ac:: Amplitude contrast: (default float)
    kboot:: kboot: (default string)
    debug:: debug mode: (default False)
    apix:: pixel size [A]: (default float)



== Output ==
The program creates the directory automatically in the micrograph directory.
