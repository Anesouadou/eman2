===== sxgui_cter =====
CTF Assessment: GUI tool to assess and sort micrographs based on their estimated CTF by [[:pipeline:cter:sxcter | sxcter]].

\\
===== Usage =====

Usage in command line

  sxgui_cter.py  cter_ctf_file 

\\
===== Typical usage =====

sxgui_cter GUI tool can be run as a stand-alone program or from the SPHIRE GUI.

  sxgui_cter.py  outdir_cter/partres.txt

\\
===== Input =====

\\
=== Main Parameters===

  ; cter_ctf_file : CTF parameter file: This file is produced by sxcter and normally called partres.txt. (default none)
